Making it easier to find the right answers

This Reading Comprehension tool enables users to pose Queries in Kannada in the context of a Kannada document. This can save users alot of their valuable time. Try this tool out for yourself to see how things can be made easier for you.

The tool was built using several Natural Language Processing techniques and was tested on a limited amount of data. The accuracy of the tool was 75%. We hope to constantly improve this tool and make it the best version it could be.
